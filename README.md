## 🛠️ Built With

- ⚛️ **React (Vite)**
- 💨 **Tailwind CSS**
- 🌐 **REST Countries API v3.1**
- 🔀 **React Router DOM**
- 🌙 **Dark/Light Mode Toggle** using Tailwind’s `dark:` variant
- 📱 **Responsive Design** for all screen sizes

---

## 💡 Features

✅ View all countries from the API on the homepage  
✅ Search for a country by name  
✅ Filter countries by region  
✅ Click on a country to see detailed information  
✅ Toggle between light and dark themes  
✅ Fully responsive layout for mobile, tablet, and desktop  

---

## 🧠 What I Learned

- How to integrate **dark mode** with Tailwind CSS using `dark:` variants.  
- Efficiently fetching and displaying data from a public API.  
- Handling route parameters using **React Router** (`useParams`).  
- Organizing React components for scalability.  
- Using conditional styling for light/dark UI themes.

## ⚙️ Installation & Setup

To run this project locally:

```bash
# 1️⃣ Clone the repository
git clone https://github.com/yourusername/rest-countries-api.git

# 2️⃣ Navigate into the folder
cd rest-countries-api

# 3️⃣ Install dependencies
npm install

# 4️⃣ Start the development server
npm run dev
```
